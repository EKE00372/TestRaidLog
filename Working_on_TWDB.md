# Working on zhTW database

不打算搞什麼成品，只會弄一份純資料來用。芒果單機的資料有兩個問題：zhTW的資料是殘廢的，而且很多翻譯都和現在的經典版不同。

做完也不會對玩家有什直接的好處，這只是用於某些資料庫插件的數據而已，所以我不會很專注在這方面。也許哪天有人用比我更好的方式把這些工作做完了呢。
